####1.什么是HTML？

HTML指的是超文本标记语言（Hyper Text Markup Language）

<b>我很粗</b>

<b></b>就是一种标记符

HTML的特点如下：

1.简易性

2.可扩展性

3.平台无关性

4.通用性

<br>

####2.HTML的标签

<h1></h1>一级标题

<hr>加横线


<p></p>段落

<br>换行

<font size = "30">font标签 </font>
